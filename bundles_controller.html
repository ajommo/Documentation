GET Bundles

Use this API to get information about all bundles that is currently available. All bundles that is currently active in the bundle-table will be returned.

Request

Path:
Headers:

GET /api/bundles/
Content-Type: application/json

Response

[
    {
        "id": "primitives",
        "name": "Primitives",
        "version": 1,
        "assets": [ 
            "cube",
            "sphere"
        ]
    }
]

GET Bundle

If only information aboud one bundle is needed, then you can get on specific bundle by providing the id for the bundle.

Request

Path:
Headers:

GET /api/bundles/{id}
Content-Type: application/json

Response

{
    "id": "primitives",
    "name": "Primitives",
    "version": 1,
    "assets": [ 
        "cube",
        "sphere"
    ]
}
